# Campus Placement Prediction using Machine Learning

A machine learning project that predicts whether a student is likely to get placed based on academic performance, internships, certifications, aptitude score, communication skills, and other relevant factors.

Project Overview

- Built a classification model to predict students' placement status.
- Performed data preprocessing and exploratory data analysis.
- Trained a Random Forest classifier for prediction.
- Evaluated the model using multiple performance metrics.

Dataset

The dataset contains information such as:

- Academic Performance
- Internships
- Projects
- Certifications
- Aptitude Score
- Communication Skills
- Placement Status (Target Variable)

Project Workflow

- Data Loading
- Data Cleaning
- Exploratory Data Analysis (EDA)
- Label Encoding
- Correlation Analysis
- Train-Test Split
- Model Training
- Model Evaluation
- Feature Importance Analysis

Machine Learning Model

- Random Forest Classifier

Libraries Used

- Python
- Pandas
- NumPy
- Matplotlib
- Seaborn
- Scikit-learn

Evaluation Metrics

- Accuracy Score
- Classification Report
- Confusion Matrix
